# Story: Ingredients Table Layout

## Problem / Context
The ingredients on the recipe page are currently displayed as a bulleted list, which makes it hard to scan and compare ingredient, quantity, and status at a glance. A table layout will better visually organize this information.

## User Story
As a recipe viewer, I want the ingredients displayed in a table so that I can more easily read and compare ingredient details.

## Scope
The table will have three columns: Ingredient, Quantity, and Status — with a header row. Existing styling (e.g. green text for "In Stock") is preserved as-is.

**Out of scope:** any changes to how status values are displayed or styled beyond what already exists.
