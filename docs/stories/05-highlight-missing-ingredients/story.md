# Story: Highlight Missing Ingredients in Red

## Problem / Context
When a user views a recipe, they need to quickly know whether they have all the ingredients on hand. Without a clear visual signal, they have to read through the availability indicator carefully. Making missing ingredients visually obvious helps the user decide at a glance whether they need to go to the store.

## User Story
As a home cook, I want missing ingredients to appear in red on the recipe page so that I can instantly tell whether I need to go to the store before making a recipe.

## Scope
**Out of scope:**
- Green or other color treatment for ingredients that are in the pantry
- Shopping list generation or any action based on missing ingredients
