# Story: About Page

## Problem / Context
Attendees of Eric Sowell's AI labs have no way to know what this application is or why it exists. Without context, they may not understand that it is a demonstration app built using Claude Code, not a production product.

## User Story
As an AI lab attendee, I want to read an about page so that I understand the purpose of this application and how to contact the presenter.

## Scope
**Out of scope:**
- Technical details about how the app was built (that lives in README.md and CLAUDE.md)
- The tech stack or AI workflow details
