# Story: Green "In Stock" Text on Recipe Page

## Problem / Context
On the recipe page, ingredient stock status text is not visually distinguished by color. Out-of-stock ingredients are already highlighted in red, but in-stock ingredients display in the default text color, making it hard to spot available ingredients at a glance.

## User Story
As a recipe viewer, I want "In stock" text to appear in green so that I can quickly identify which ingredients I have available without reading every line carefully.

## Scope
**Out of scope:** Adding or changing any "out of stock" visual treatment (already exists).
