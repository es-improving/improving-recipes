# Story: Test Page

## Problem / Context
The developer needs a simple, stable page to trigger and verify GitHub Actions workflows without touching production features.

## User Story
As a developer, I want a /test page in the navigation so that I have a predictable target for validating GitHub Actions behavior.

## Scope
**Out of scope:**
- Any real content or functionality on the page beyond the static text
- This page is not intended for end users

## Open Questions
None.
